# 🧾 User Information Python Program

## 📌 Overview

This is a simple Python program that collects user information such as name, age, height, and favorite number. It then performs basic processing like type conversion and calculates the user's birth year.

The program also displays the data type and memory address of each variable for learning purposes.

---

## 🚀 Features

* Takes user input from the console
* Converts input values into appropriate data types
* Calculates birth year based on current year
* Displays:

  * Value
  * Data type
  * Memory address (using `id()`)

---

## 🛠️ Technologies Used

* Python 3

---

## 📥 How to Run

1. Make sure Python is installed on your system.
2. Copy the code into a file named `user_info.py`
3. Open terminal or command prompt
4. Run the program:

```bash
python user_info.py
```

## ⚠️ Notes

* The program assumes valid numeric input for age, height, and favorite number.
